## 1. Простой Калькулятор

**Описание:**  
Простой калькулятор с графическим интерфейсом на Tkinter. Поддерживает:

- Основные арифметические операции: +, -, *, /  
- Квадратный корень  
- Обработку ошибок деления на ноль и некорректного ввода  
- Валидацию: нельзя вводить буквы в поле ввода

**Файл:** `calculator.py`

**Запуск:**
```bash
python calculator.py
````

**Пример работы:**

* Ввод чисел и операций через кнопки
* Кнопка `=` для расчёта результата
* Кнопка `C` для очистки
* Кнопка `√` для извлечения квадратного корня

## 2. Учёт пациентов (Patient Management App)

**Описание:**
Программа для хранения и анализа информации о пациентах с визуализацией статистики.

**Возможности:**

* Добавление и редактирование пациентов
* Ввод данных: ФИО, возраст, пол, рост, вес
* Автоматический расчёт ИМТ (BMI)
* Визуализация статистики:

  * Распределение пола
  * Возраст
  * Индекс массы тела (BMI)
  * Зависимость ИМТ от возраста

**Файл:** `patients.py`

**Запуск:**

```bash
python patients.py
```

**Особенности:**

* Используется `dataclass` для хранения данных пациента
* Данные сохраняются в `patients.json`
* Статистика строится с помощью `matplotlib`
* Таблица с пациентами обновляется автоматически
* GUI разделён на основной интерфейс и отдельную форму пациента

**Пример интерфейса:**

* Кнопка `Добавить` открывает форму нового пациента
* Кнопка `Редактировать` позволяет изменять выбранного пациента
* Кнопка `Статистика` строит графики распределения данных

## 3. FastQC Lite

**Описание:**
Мини-версия FastQC для анализа качества FASTQ-файлов. Поддерживает:

* Чтение FASTQ и FASTQ.gz файлов
* Построение графиков:

  1. Per base sequence quality (среднее качество по позиции)
  2. Per base sequence content (процентное содержание A/C/G/T)
  3. Sequence length distribution (распределение длин ридов)
* Прогресс-бар для больших файлов
* Ограничение по количеству ридов для ускорения работы (MAX_READS)

**Файлы:**

* `fastq.py` — GUI и построение графиков
* `fastq_reader.py` — класс для чтения FASTQ
* `record.py` — класс SequenceRecord

**Запуск:**

```bash
python fastq.py
```

**Особенности:**

* Графики построены с помощью `matplotlib`
* Максимально простой и читаемый код
* Поддержка файлов сжатых в `.gz`

**Пример графиков:**

* Линия качества по позициям нуклеотидов
* Процентное содержание A/C/G/T по позициям
* Распределение длин ридов
